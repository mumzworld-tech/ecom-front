:pill: Mumzworld Ecom Frontend Test Project

## Getting Started

-   Clone the repository
-   Create .env based on .env.example
-   Run

```bash
yarn install
yarn dev
```

-   Run `bash yarn generate` if you need to update graphql types

## Stack

-   NextJs
-   Tailwind
-   Apollo Graphql Client
-   Graphql Code Generator
-   headlessui/react
